Sobre Servidores

[Voltar]

Os servidores são computadores remotos nos quais as informações que as aplicações necessitam ficam armazendas, também são usados para processamento quando a aplicação demanda alta velocidade de acesso e manipulão de grande quantidades de dados.

Em informática, um servidor é um software ou computador, com sistema de computação centralizada que fornece serviços a uma rede de computadores, chamada de cliente. Esses serviços podem ser de naturezas distintas, como por exemplo, arquivos e correio eletrônico. Esta arquitetura é chamada de modelo cliente-servidor, é utilizada em redes de médio e grande porte (com muitas máquinas) e em redes onde a questão da segurança desempenha um papel de grande importância. O termo servidor é amplamente aplicado a computadores completos, embora um servidor possa equivaler a um software ou a partes de um sistema computacional, ou até mesmo a uma máquina que não seja necessariamente um computador. Os servidores podem fornecer várias funcionalidades, muitas vezes chamado de "serviços", tais como a partilha de dados ou de recursos do sistema entre vários clientes, ou computação desempenho para um cliente. Um único servidor pode servir vários clientes, e um único cliente pode usar vários servidores. Um processo cliente pode ser executado no mesmo dispositivo ou pode se conectar através de uma rede para um servidor em um dispositivo diferente.[Wikipedia]

Estrutura Básica do HTML

[Voltar]

A seção sobre estrutura informou sobre as partes de um documento HTML, como por exemplo, que este deve ser iniciado pela tag <html> e finalizado pela tag </html>.

Após iniciar o documento com a tag indicada, passa-se para o cabeçalho, iniciado e encerrado pelas tags <head> e </head>, respectivamente, entre essas tags serão informados alguns parametros necessários encerrados entre as tags <meta> e </meta>

Ainda dentro do cabeçalho, é necessário especificar um título para a página, essa informação aparecerá como o nome da aba no navagador do usuário, para isso deve-se inserir o título entre as seguintes tags: <title> e </title>

Em sequência, inicia-se a seção do "corpo" do documento, que é a parte onde efetivamente se encontrarão as informações que serão mostradas ao usuário da página, para formatar o texto é necessário que se use todo um leque de tags que serão tratadas nas próximas seções, as informações do corpo ficam entre as tags <body> e </body>.

Juntando todas essas informações o "esqueleto" do documento ficará, basicamente, com a seguinte aparência: 

    
        <html>
    
            <head>
                <meta  A /> 

                <meta  B  />

                <title>...Título do Documento...</title>
            </head>
        
            <body>
                ...Informações que serão mostradas aos usuários...
            </body>
        
        </html>

Listas Ordenadas e Não Ordenadas

[Voltar]

É possível criar listas de dois tipos, ordenada e não ordenada, a primeira é aquela em que seus itens precisam respeitar uma ordem específica, por isso, são marcados com numeração arábica, romana ou letras. Já o segundo tipo não demanda uma ordem específica, portanto, seus itens podem ser destacados apenas por símbolos sem conotação de ordem com bolinhas, circulos, quadrados, etc.

LISTA ORDENADA:

Para criar uma lista ordenada, utiliza-se a tag <ol> (ordered list), e para especificar cada item utiliza-se a tag <li> (list item), segue exemplo: 

    
    <ol>
        <li>Primeiro item</li>
        <li>Segundo item</li>
        <li>Terceiro item</li>
    </ol>

O resultado será:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Como já informado é possível utlizar outros símbolos de ordenamento, como letras ou números romanos, utilizando o atributo type de dentro da tag <ol> um exemplo seria: 

    
    <ol type="A">
        <li>Primeiro item</li>
        <li>Segundo item</li>
        <li>Terceiro item</li>
    </ol>

O resultado será:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Com o atributo type é possível utilizar as opções:

  • "A" : Para letras maiúsculas;
  • "a" : Para letras minúsculas;
  • "i" : Para algarismos romanos minúsculos;
  • "I" : Para algarismos romanos maiúsculos;
LISTA NÃO ORDENADA:

A lista não ordenada contém itens que não dependem de uma ordem específica, dessa forma não precisam de numerais ou letras sequenciais. Para criar uma lista ordenada, utiliza-se a tag <ul> (unordered list), e para especificar cada item utiliza-se a tag <li> (list item), segue exemplo: 

    
    <ul>
        <li>Primeiro item</li>
        <li>Segundo item</li>
        <li>Terceiro item</li>
    </ul>

O resultado será:

  • Primeiro item
  • Segundo item
  • Terceiro item

O atributo type também se aplica para a lista não ordenada, no entanto as opções são:

  • circle:
    • Círculo
  • disc:
    • Disco
  • square:
    • Quadrado
  • triangle:
    • Triangulo (Não são todos os navegadores que tem suporte à esta opção)

Links

[Voltar]

No texto pode ser importante inserir links que levem para páginas externas ou outras partes do próprio documento, estes são links externos e de ancoragem, respectivamente. A tag para criar links é <a>, exemplos de utilização:

Para link externo:
    
    <a href="https://www.google.com" target ="_blank">clique aqui.</a>

Dessa forma, no texto aparecerá apenas a frase clique aqui, ao clicar o usuário será redirecionado para a página especificada, nesse exemplo seria para "https://www.google.com" em uma nova aba, devido ao atributo target ="_blank". Existem várias outras opções de atributos que permitem a interação com midias, sites, endereços de e-mail e até com scripts em javascript.

Para ancoragem:

Para definição do ponto de ancoragem utilize:

            
    <a name="nome_do_ponto_de_ancoragem"></a>

Para definir o ponto de retorno em referência ao ponto de ancoragem utilize da seguinte forma: 

    
    <a href="#nome_do_ponto_de_ancoragem">voltar</a>

Assim, no texto, aparecerá apenas a palavra voltar e, quando o usuário clicar nela, o texto voltará para o ponto onde o "nome_do_ponto_de_ancoragem" foi definido.
